AuraShellWin11 风格 SSH 终端

一个跨平台的SSH客户端, 具有远程文件管理器和集成终端, 提供现代化、优雅且高效的远程管理体验.

核心功能

🎨

Win11 风格用户界面

采用 QFluentWidgets 实现 Fluent Design 风格, 支持亮/暗主题无缝切换.

🖥️

集成 SSH 终端

基于 xterm.js, 支持命令行交互、复制粘贴、并可自定义字体和配色方案.

📂

远程文件管理器

支持文件上传、下载、重命名、删除和权限修改, 提供类似 Windows 资源管理器的体验.

⚡

多会话管理

支持同时进行多个远程连接, 并在不同会话之间轻松切换, 会话信息本地持久化.

📊

系统资源监控

实时显示远程主机的 CPU 和内存使用率, 并可查看资源占用最高的进程列表.

🛡️

多种认证方式

支持密码和 SSH 密钥两种认证方式, 确保连接的灵活性和安全性.

📦

打包压缩传输

支持将多个文件或文件夹打包成压缩文件进行传输, 提高效率并减少带宽占用.

☁️

Gist 云同步

与 GitHub Gist 集成, 可将常用脚本和配置文件同步到云端, 方便多设备访问.

🔒

SFTP 协议支持

基于安全的 SFTP 协议进行文件传输, 确保数据在传输过程中的完整性与机密性.
